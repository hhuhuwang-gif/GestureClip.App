## 本次修复（建议升级）

- **`Ctrl+Shift+V` 纯文本粘贴** 无效：先释放 Shift/Ctrl 等修饰键再注入粘贴。
- **剪贴板历史 Enter 粘贴** 无效：先隐藏面板恢复原窗口焦点，再粘贴。
- 智能粘贴 / 无文本回退；写剪贴板后等待落定。
