# GestureClip v0.6.30 Beta

## 快捷键

设置 → 剪贴板 → 快捷键：

1. 点右侧「录制」或点灰色框  
2. 按下 **Ctrl/Alt/Shift/Win + 主键**  
3. 看到状态提示「已设置 / 已保存」即成功  
4. Esc 取消，Backspace 恢复默认  

## 安装包

- `GestureClip-v0.6.30-beta-win-x64.zip`
- `GestureClip-Setup-v0.6.30-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 剪贴板

历史支持「已复制」与筛选。

## 数据

`%LOCALAPPDATA%\GestureClip\` 覆盖安装不删除。
