# GestureClip v0.6.4 Beta

## 下载

- `GestureClip-v0.6.4-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 这次更新

- 修复检查更新在本机代理端口不可用时失败的问题。
- 如果系统代理例如 `127.0.0.1:7890` 没开，检查更新和下载更新会自动直连重试。
- 优化错误提示：不再把“目标计算机积极拒绝 / 127.0.0.1:7890”这种吓人的底层错误直接甩给用户。
- 保留“打开 GitHub Release 页面手动查看”的兜底入口。

## 更新流程

1. 点击「检查更新」。
2. 程序优先按系统网络设置访问 GitHub。
3. 如果本机代理不可用，会自动直连重试。
4. 发现新版本后可继续一键覆盖更新。

## 本地数据说明

覆盖程序文件不会删除本地历史记录和设置；默认数据保存在：

```text
%LOCALAPPDATA%\GestureClip
```

## 校验

下载后可使用 `SHA256SUMS.txt` 校验 zip 文件完整性。

## 已知说明

- 当前 Beta 包未签名，Windows SmartScreen 可能提示未知发布者。
- 管理员权限窗口中，普通权限运行的 GestureClip 可能无法稳定执行输入模拟。
