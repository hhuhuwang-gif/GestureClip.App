## 本版更新（设置界面高保真重设计）

- **三栏布局**：左侧分组导航（常用 / 系统 / 其他）、中央全宽设置、可折叠右侧状态栏
- **玻璃浅色主题**：snow / cloud / slate / sky-blue，减少框套框与视觉噪声
- **可用设置搜索**：搜索并滚动到具体分组（智能粘贴、快捷键、左键增强等）
- **手势 / 动作绑定页扁平化**：分区标题 + 列表容器；动作下拉选择更可靠
- **窗口可缩放 + 细滚动条贴边**：无边框窗口拖边调整
- **开关样式**：系统设置风格胶囊开关

兼容 v0.6.24 数据与本地路径。

已复制绿色标记条等能力仍可用。
