# GestureClip v0.6.26 Beta

## 亮点

- 手势配置 **导入 / 导出**（JSON）
- 一键场景模板：办公 / 浏览 / 剪贴板
- 首次引导跳转与文案加强
- **插入日期/时间**与**常用话术 1/2/3**（可编辑）
- 设置 UI 重设计、热键点击录制、深色高对比、窗口记忆
- Setup 安装器流水线与可选代码签名脚本

## 安装包

- `GestureClip-v0.6.26-beta-win-x64.zip`
- `GestureClip-Setup-v0.6.26-beta-win-x64.zip`（解压后双击 Setup.cmd）
- 校验：`SHA256SUMS.txt`

## 安装 / 覆盖更新

首次：解压 Setup zip → Setup.cmd，或便携 zip 后双击 `GestureClip.exe`。  
覆盖：先退出程序再覆盖；用户数据在 `%LOCALAPPDATA%\GestureClip\`，覆盖安装不删除。

剪贴板历史支持「已复制」与筛选；双击复制等交互与 v0.6.x 一致。
