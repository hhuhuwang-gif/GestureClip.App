# GestureClip v0.6.33 Beta

## 推荐安装（便携）

1. 下载 `GestureClip-v0.6.33-beta-win-x64.zip`
2. 解压后双击 `GestureClip.exe`
3. 数据在 `%LOCALAPPDATA%\GestureClip\`

也可使用 Setup zip → `Setup.cmd`。

## 本版亮点

- **手势 HUD**：`手势 · 复制 · 粘贴 · 少点` 一行高对比，不再看不清
- 含 0.6.32 工位小熊 Hub 美化
- 含 0.6.31 OCR / 托盘 / 热键冲突提示

## 安装包

- `GestureClip-v0.6.33-beta-win-x64.zip`（推荐）
- `GestureClip-Setup-v0.6.33-beta-win-x64.zip`
- `SHA256SUMS.txt`

历史支持「已复制」、固定、收藏；定时清理不删固定/收藏。
