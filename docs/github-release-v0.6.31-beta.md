# GestureClip v0.6.31 Beta

## 推荐安装（便携）

1. 下载 `GestureClip-v0.6.31-beta-win-x64.zip`
2. 解压后双击 `GestureClip.exe`
3. 数据在 `%LOCALAPPDATA%\GestureClip\`

也可使用 Setup zip → `Setup.cmd`（不要点 .txt）。

## 本版亮点

- 托盘：剪贴板/手势开关、暂停全部、卸载入口
- 热键冲突提示 + 可尝试组合
- 图片 OCR（本地）可被历史搜索命中
- 启动更快（次要服务后台启动）
- 含 0.6.30 快捷键录制修复

## 安装包

- `GestureClip-v0.6.31-beta-win-x64.zip`（推荐）
- `GestureClip-Setup-v0.6.31-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 剪贴板

历史支持「已复制」、固定、收藏；定时清理不删固定/收藏。

## 数据

`%LOCALAPPDATA%\GestureClip\` 覆盖安装不删除。
