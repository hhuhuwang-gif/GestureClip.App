# GestureClip v0.6.3 Beta

## 下载

- `GestureClip-v0.6.3-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 这次更新

- 新增“检查更新”功能。
- 设置页「检查更新」会比较当前版本与 GitHub Latest Release。
- 托盘菜单也新增「检查更新」。
- 如果当前已是最新版本，会明确提示当前版本和最新版本。
- 如果发现新版本，会提示是否立即下载并覆盖安装。
- 自动更新能力拆分为“检查更新”和“一键覆盖更新”，后续更容易扩展启动后静默检查、忽略版本和更新说明展示。

## 更新流程

1. 点击「检查更新」。
2. 程序读取 GitHub Latest Release。
3. 和当前程序版本比较。
4. 如果发现新版本，可直接点击确认进入一键覆盖更新。
5. 覆盖更新会下载 zip、退出当前程序、覆盖旧版并重启。

## 本地数据说明

覆盖程序文件不会删除本地历史记录和设置；默认数据保存在：

```text
%LOCALAPPDATA%\GestureClip
```

## 校验

下载后可使用 `SHA256SUMS.txt` 校验 zip 文件完整性。

## 已知说明

- 当前 Beta 包未签名，Windows SmartScreen 可能提示未知发布者。
- 管理员权限窗口中，普通权限运行的 GestureClip 可能无法稳定执行输入模拟。
- 覆盖更新前必须先退出程序，否则 `GestureClip.exe` 可能被占用。
