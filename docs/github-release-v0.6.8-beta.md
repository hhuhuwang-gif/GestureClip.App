# GestureClip v0.6.8 Beta

## 下载

- `GestureClip-v0.6.8-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 本次重点

- 手势设置页继续优化：常用手势和高级手势列表支持就近删除，不再只能到动作编辑器里删除。
- 删除自定义手势前会先弹出确认，避免误触导致已配置的手势丢失。
- 手势列表选中项新增“当前选中”提示，并继续使用浅色自定义选中态，减少系统默认黑块高亮问题。
- 项目新增 `.editorconfig` 和 `.gitattributes`，减少后续编辑时出现行尾空格、换行污染和二进制文件误识别。

## 延续优化

- 保留上一版剪贴板历史双击复制长文本的稳定性修复：长文本复制更快，不易卡顿或截断。
- 保留剪贴板历史双击记录后自动复制到系统剪贴板并关闭面板的体验。
- 保留手势添加的二次确认体验，避免误触直接保存。

## 使用提示

1. 进入「设置」→「鼠标右键手势」。
2. 在常用手势或高级手势列表中选中已有手势，可以直接在列表附近删除。
3. 删除前会看到确认提示，确认后才会移除本机自定义手势。
4. 在剪贴板历史里双击任意记录：复制到系统剪贴板并自动关闭面板。
5. 需要直接粘贴时，按 Enter 或点击“粘贴”。复制按钮只复制，不会关闭面板。

## 本地数据说明

覆盖程序文件不会删除本地历史记录和设置；默认数据保存在：

```text
%LOCALAPPDATA%\GestureClip
```
