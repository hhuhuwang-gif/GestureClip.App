# GestureClip v0.6.5 Beta

## 下载

- `GestureClip-v0.6.5-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 这次更新

- 优化鼠标手势设置页：新建自定义手势时，第一次点击会进入“确认添加”，第二次点击才真正保存，避免误触把未确认的手势写入列表。
- 选择手势动作、修改手势码或重新画手势后，会自动取消上一次确认状态，减少误操作。
- 优化手势列表和动作下拉框的选中态：改为浅蓝高亮，不再出现突兀的黑色选中块。
- 剪贴板历史支持双击记录后自动复制到系统剪贴板并关闭面板；Enter 和“粘贴”按钮仍保留粘贴并关闭。
- 顶部说明补充“复制按钮只复制不关闭”，减少误解。
- 复制成功提示改成“文字/图片已复制到系统剪贴板”，结果更明确。
- 删除剪贴板历史记录前会先确认，并说明只删除本机历史，不影响当前系统剪贴板内容。
- 修复手势执行“粘贴 / 粘贴并回车”时没有累计粘贴次数的问题，复制/粘贴统计现在更准确。
- 剪贴板历史列表选中态也改为浅蓝高亮，视觉更统一。
- 保留上一版的手势模板、Del 删除入口和添加后自动选中新手势体验。

## 使用提示

1. 在剪贴板历史里双击任意记录：复制到系统剪贴板并自动关闭面板。
2. 需要直接粘贴时，按 Enter 或点击“粘贴”。复制按钮只复制，不会关闭面板。
3. 进入「设置」→「鼠标右键手势」。
4. 在「新建自定义手势」里画手势或点模板。
5. 选择动作后，先点「添加到手势列表」。
6. 按钮变成「确认添加到手势列表」后，再点一次完成保存。

## 本地数据说明

覆盖程序文件不会删除本地历史记录和设置；默认数据保存在：

```text
%LOCALAPPDATA%\GestureClip
```
