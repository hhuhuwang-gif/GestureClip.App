# GestureClip v0.6.9 Beta

## 下载

- `GestureClip-v0.6.9-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 本次修复

- 修复点击「动作绑定」页或点击手势列表后程序可能卡死的问题。
- 常用手势列表和高级手势列表不再共用同一个 WPF `SelectedItem` 绑定，避免一个列表选中后另一个列表回写空选择，导致编辑器选中状态反复拉扯。
- 动作绑定列表开启虚拟化回收，降低进入页面、切换手势和刷新列表时的 UI 重绘压力。

## 延续优化

- 保留剪贴板历史双击记录后自动复制到系统剪贴板并关闭面板的体验。

## 使用提示

1. 打开「设置」→「动作绑定」。
2. 点击常用手势或高级手势，右侧/下方动作编辑器应立即显示当前动作，不应卡死。
3. 修改动作后会自动保存；删除手势仍会先确认。

## 本地数据说明

覆盖程序文件不会删除本地历史记录和设置；默认数据保存在：

```text
%LOCALAPPDATA%\GestureClip
```

## 校验

下载后可使用 `SHA256SUMS.txt` 校验 zip 文件完整性。

## 已知说明

- 当前 Beta 包未签名，Windows SmartScreen 可能提示未知发布者。
- 管理员权限窗口中，普通权限运行的 GestureClip 可能无法稳定执行输入模拟。
