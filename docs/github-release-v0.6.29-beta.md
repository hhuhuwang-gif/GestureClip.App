# GestureClip v0.6.29 Beta

## 安装说明（重要）

1. 下载 **Setup zip** 并解压  
2. 双击 **`Setup.cmd`**（不要点任何 .txt）  
3. 黑窗口显示进度后自动安装并启动  

若你之前版本弹出记事本，是因为包内曾附带「安装说明.txt」，双击它会打开文本——**不是程序异常**。0.6.29 起已移除该文件。

## 亮点

- 安装器不再附带易误点的 .txt  
- 含 v0.6.28 设置文字可见性修复等

## 安装包

- `GestureClip-v0.6.29-beta-win-x64.zip`
- `GestureClip-Setup-v0.6.29-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 数据

`%LOCALAPPDATA%\GestureClip\` 覆盖安装不删除。

## 剪贴板

历史支持「已复制」与筛选。
