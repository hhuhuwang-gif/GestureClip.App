## 本次重点

- 修复 ChatGPT / 浏览器富文本复制后，再从 GestureClip 历史复制时可能不是完整正文的问题。
- 当剪贴板同时包含 Unicode 文本、HTML、图片快照时，优先保存可用文本，只在没有文本时才保存图片。
- 搜索结果、列表项、双击复制、复制按钮和粘贴动作统一加载完整正文，避免只写入预览文本。
- 保留剪贴板历史双击记录后自动复制并关闭的操作习惯。
- 动作绑定页改成更小白友好的卡片式结构：手势方向、中文名称、当前动作、说明和就近删除。
- 新增“推荐给新手的 3 个手势”，一键补齐缺失推荐项，不删除、不覆盖已有自定义手势。
- 设置页统一入口说明、按钮宽度、表单行距和危险操作区样式。

## 使用提示

1. 真实网页复制 ChatGPT 回复后，可从剪贴板历史双击或点复制验证完整正文。
2. 进入「设置 → 动作绑定」，可以先点“一键添加推荐手势”快速开始。
3. 危险操作现在集中显示在浅红区域，执行前请看清影响范围。
