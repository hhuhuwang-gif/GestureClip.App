# GestureClip v0.5.0 Beta - 发布包与覆盖更新验证版

## 本次重点更新

- 剪贴板文本 / 图片历史体验优化。
- 图片缩略图加载策略优化。
- 快速复制、搜索和列表滚动性能优化。
- 手势绑定页面精简。
- 自定义手势入口优化。
- 工位小熊 HUD 优化。
- 新增过劳提醒 / 猝死提醒。
- HUD 支持根据工作时间阶段动态变色。
- HUD 可读性增强。
- 退出流程优化，降低后台残留风险。
- 发布包和覆盖更新流程整理。

## 下载文件

- `GestureClip-v0.5.0-beta-win-x64.zip`：新用户完整包，解压后运行 `GestureClip.exe`。
- `GestureClip-v0.5.0-beta-update-win-x64.zip`：老用户覆盖更新包。
- `SHA256SUMS.txt`：压缩包 SHA256 校验值。

## 覆盖更新方法

1. 右键托盘图标，选择“退出 GestureClip”。
2. 打开任务管理器，确认没有 `GestureClip` 进程。
3. 解压 update 压缩包。
4. 将新版文件覆盖到旧版程序目录。
5. 双击 `GestureClip.exe` 启动新版。

## 数据保留说明

覆盖更新不会删除用户数据。默认数据目录：

```text
%LOCALAPPDATA%\GestureClip\
```

会保留：

- 剪贴板历史
- 图片历史
- 常用片段
- 手势配置
- 自定义手势
- 黑名单
- HUD 设置
- 工位小熊设置
- 猝死提醒设置
- 打工人等级 / XP
- 用户设置

## 已知问题

- 当前 Beta 包未签名，Windows SmartScreen 可能提示未知发布者。
- 管理员权限窗口中，普通权限运行的 GestureClip 可能无法稳定执行输入模拟。
- 图片历史数量很大时，首次缩略图加载仍可能需要一点时间。
