## 本次更新

### UI 美化

- 明/暗主题渐变与卡片色统一
- 剪贴板历史面板、列表卡片、分段筛选更精致
- 设置页 / 快捷动作外壳阴影与边框统一
- 手势 HUD 轨迹与面板质感微调

### 稳定性

- 手势粘贴记住目标窗口，减少「滑了没贴上」
- 历史再复制统一纯文本（网页/ChatGPT 不易半残）
- 剪贴板读取多路径兜底
- 防误触默认阈值略增
- 多选「已复制」绿色色条标记仍可用

## 数据

`%LOCALAPPDATA%\GestureClip\` 覆盖程序不会删除。
