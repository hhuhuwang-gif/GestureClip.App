# GestureClip v0.6.14 Beta

## 下载

- GestureClip-v0.6.14-beta-win-x64.zip
- SHA256SUMS.txt

## 本次重点

- 新增本地「快捷动作」面板（默认 `Ctrl + Shift + Q`）：去空格、大小写、JSON 美化/压缩、URL 编解码等，不上传。
- 手势可绑定文本处理动作；设置里可改快捷动作热键。
- 左键增强动作可配置（按住右键画手势时再点左键）。
- 工位小熊 Hub 分区折叠 + 30 秒开工配置；休息提醒支持日/周上限与连续工作阈值。
- 托盘可看今日已赚摘要、开关下班冲刺、生成本周总结。
- 分享卡片支持多种样式，生成后可打开文件夹；支持本地周报/月报。
- UI 更接近 Apple 风格；修复蓝色主按钮文字看不清；交通灯：红关 / 黄放大 / 绿缩小。
- 检查更新会显示更新说明；关于页可看本地 CHANGELOG。

## 使用提示

1. 复制一段文字后按 `Ctrl + Shift + Q` 试 JSON 美化或去空格。
2. 托盘 → 工位小熊 → 打开 Hub，完成月薪与上下班时间引导。
3. 设置 → 动作绑定 → 左键增强动作，可改「下划 + 左键」等增强绑定。
4. 关于页可「检查更新」并查看更新日志。

## 本地数据

- 覆盖更新会保留 `%LOCALAPPDATA%\GestureClip` 下的历史、设置与日志。
- 诊断包不包含剪贴板正文与图片原图。
