# GestureClip v0.6.27 Beta

## 亮点

- **智能粘贴应用规则**：按进程覆盖 Plain / Clean / Normal，支持用前台窗口一键添加
- **屏蔽名单**：不记剪贴板 / 禁用手势分项；默认名单扩展
- **代码签名文档与 CI**：配置 secrets 后可真签名（见 docs/CODE_SIGNING.md）
- **30 秒图文演示**：docs/DEMO_30S.md
- 含 v0.6.26：导入导出、模板、话术/日期、热键录制、Setup 包

## 安装包

- `GestureClip-v0.6.27-beta-win-x64.zip`
- `GestureClip-Setup-v0.6.27-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 剪贴板

历史支持「已复制」与筛选；双击复制等交互与 v0.6.x 一致。

## 数据

`%LOCALAPPDATA%\GestureClip\` 覆盖安装不删除。

## 签名

未配置证书时包可能未签名。配置方法见 [CODE_SIGNING.md](CODE_SIGNING.md)。
