# GestureClip v0.6.32 Beta

## 推荐安装（便携）

1. 下载 `GestureClip-v0.6.32-beta-win-x64.zip`
2. 解压后双击 `GestureClip.exe`
3. 数据在 `%LOCALAPPDATA%\GestureClip\`

也可使用 Setup zip → `Setup.cmd`（不要点 .txt）。

## 本版亮点

- 工位小熊 Hub：暗色可读、阶段色、进度条、玻璃分区
- 空状态月薪引导、重置二次确认、Esc 关闭
- 摸鱼进行中提示、日报分段排版、轻淡入
- 含 0.6.31：OCR、托盘开关、热键冲突提示、启动加速

## 安装包

- `GestureClip-v0.6.32-beta-win-x64.zip`（推荐）
- `GestureClip-Setup-v0.6.32-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 剪贴板

历史支持「已复制」、固定、收藏；定时清理不删固定/收藏。

## 数据

`%LOCALAPPDATA%\GestureClip\` 覆盖安装不删除。
