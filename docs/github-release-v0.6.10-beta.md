# GestureClip v0.6.10 Beta

## 下载

- `GestureClip-v0.6.10-beta-win-x64.zip`
- `SHA256SUMS.txt`

## 本次重点

- 项目目标明确为：把 GestureClip 做成“小白也敢用、办公党每天离不开的 Windows 本地办公加速器”。
- 新增产品目标、自优化提示词和 v1.0 路线图，后续每次优化都按稳定、小白友好、本地优先、发布闭环来验收。
- README 首屏改成“30 秒上手”，新用户可以直接照着使用。
- 设置首页新增“30 秒上手”卡片，打开软件即可看到剪贴板历史、双击找回内容、右键手势和本地优先说明。

## 延续优化

- 保留剪贴板历史双击记录后自动复制到系统剪贴板并关闭面板的体验。
- 保留 v0.6.9 对动作绑定页卡死问题的修复。

## 使用提示

1. 打开 GestureClip 后进入「设置」首页。
2. 先看“30 秒上手”卡片：`Ctrl + \`` 打开剪贴板历史，双击记录找回内容，右键划一下试手势。
3. 想改手势动作：进入「设置」→「动作绑定」。

## 本地数据说明

覆盖程序文件不会删除本地历史记录和设置；默认数据保存在：

```text
%LOCALAPPDATA%\GestureClip
```

## 校验

下载后可使用 `SHA256SUMS.txt` 校验 zip 文件完整性。
