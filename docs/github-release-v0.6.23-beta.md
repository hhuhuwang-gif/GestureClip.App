# GestureClip v0.6.23 Beta

## 下载

- [GestureClip-v0.6.23-beta-win-x64.zip](https://github.com/hhuhuwang-gif/GestureClip.App/releases/download/v0.6.23-beta/GestureClip-v0.6.23-beta-win-x64.zip)
- SHA256SUMS.txt

解压后双击 `GestureClip.exe`。

## 本次更新（设置更人性化）

- 点「更换动作」直接跳到修改区
- 手势页顶部可去设计动作 / 查看边缘增强 / 展开高级设置
- 屏幕边缘增强卡片默认可见（不再只藏在折叠里）
- 首页功能介绍卡片可点击跳转
- 新人欢迎页点卡片进入对应设置

仍包含 v0.6.22 的 UI 美化与粘贴稳定性改进。

多选「已复制」绿色色条标记仍可用。

## 数据

`%LOCALAPPDATA%\GestureClip\` 覆盖程序不会删除。
