## 本次重点

- 修复「设置 → 动作绑定」点击后卡死 / 未响应的问题。
- 将动作绑定页改成更稳定的轻量渲染，不再一次性渲染复杂 ListBox 卡片。
- 常用手势改成三列等宽网格，一眼能看到有哪些手势。
- 修复手势卡片歪斜、没对齐、文字显示不完整的问题。
- 修复下拉框显示 `GestureBindingCardViewModel` 类型名的问题。
- 恢复设置窗口空白区域拖动，同时避免按钮、下拉框、滚动区域误触拖动。

## 使用提示

进入「设置 → 动作绑定」后：

1. 直接点击常用手势卡片。
2. 在下方修改动作。
3. 自定义手势仍在页面下方创建。
