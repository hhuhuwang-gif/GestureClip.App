# CHANGELOG

## GestureClip v0.6.0 Beta

### 重点更新

- 工位小熊升级为办公状态 Hub：顶部实时状态卡、今日收益与效率、摸鱼与休息、下班冲刺、今日报告。
- 新增手动摸鱼模式，记录本次/今日摸鱼时长与娱乐估算价值。
- 新增下班后自动报告，每天最多弹出一次，支持今天不再显示。
- 新增 PNG 分享卡，默认保存桌面，不上传、不自动分享、不包含剪贴板内容。
- HUD 联动工位状态，支持阶段变色和高频“复制 +5”合并提示。
- 托盘菜单新增工位小熊子菜单：Hub、摸鱼、报告、分享卡、休息提醒。
- 剪贴板历史新增“常显”按钮：默认失焦自动隐藏，选择常显后才置顶常驻。
- 设置页同步新增自动报告、摸鱼、下班冲刺、分享卡、HUD 状态文案、HUD 阶段变色等开关。
- 工位小熊 Hub 数据库迁移改为可重复检查，增强旧库恢复能力。

### 隐私与性能

- 报告和分享卡不包含剪贴板正文、图片内容、浏览器内容或敏感路径。
- 工位小熊统计全部本地保存，不上传。
- HUD 使用短缓存，托盘菜单避免同步阻塞，分享卡后台生成。

### 验证

- dotnet restore 通过。
- dotnet build 通过，0 警告 0 错误。
- dotnet test 通过，459 passed。
## GestureClip v0.5.1 Beta

### 重点更新

- 升级发布版本为 `0.5.1-beta`。
- 优化自定义手势动作选择区域，减少拥挤和误点。
- 优化手势绑定页滚动与右侧动作编辑器常显体验。
- 增强自定义手势识别稳定性。
- 默认补充 R+L：右键按住 + 左键点击组合手势。
- 增加“导出诊断包”，便于 Beta 用户反馈问题。
- 补充 README、HELP、UPDATE、BETA_TEST、KNOWN_ISSUES 文档。
- 新增 GitHub Bug / 功能建议 Issue 模板。

### 修复

- 修复手势绑定页左侧列表滚轮体验异常的问题。
- 修复右侧动作编辑器被滚走后出现大面积空白的问题。
- 修复自定义手势入口按钮点击无效的问题。
- 修复自定义手势添加后不立即显示的问题。
- 修复设置页自定义手势识别阈值过高导致识别失败的问题。

### 已知问题

- 当前 Beta 包未签名，Windows SmartScreen 可能提示未知发布者。
- 管理员权限窗口中，普通权限运行的 GestureClip 可能无法稳定执行输入模拟。
- 图片历史数量很大时，首次缩略图加载仍可能需要一点时间。
- 覆盖更新前必须先退出程序，否则 `GestureClip.exe` 可能被占用。

## GestureClip v0.5.0 Beta

- 优化剪贴板快速点击复制的流畅度。

- 优化退出流程，减少后台残留风险。

- 修复部分场景退出后可能残留进程的问题。
- 修复 HUD 浅色背景下文字看不清的问题。
