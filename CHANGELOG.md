# CHANGELOG

## GestureClip v0.6.8 Beta

### 重点更新

- 优化手势设置页：常用手势和高级手势列表支持就近删除，不再只能到动作编辑器里删除。
- 删除手势前新增确认弹窗，避免误触导致自定义手势丢失。
- 手势列表选中项新增“当前选中”提示，并继续沿用浅色自定义选中态，减少系统默认黑块高亮问题。
- 新增 `.editorconfig` 和 `.gitattributes`，降低后续编辑时出现行尾空格、换行污染和二进制文件误识别的风险。

### 验证

- `git diff --check`
- `dotnet test ./GestureClip.sln`
- `scripts/publish-win-x64.ps1`

## GestureClip v0.6.5 Beta

### 重点更新

- 手势设置页新增“二次确认添加”：新建自定义手势时，第一次点击只进入确认态，第二次点击才保存，避免误触。
- 修改手势码、重新画手势或更换动作后，会自动取消之前的确认态。
- 手势列表和动作下拉框的选中态改为浅蓝高亮，不再出现突兀黑块。
- 剪贴板历史支持双击记录后自动复制到系统剪贴板并关闭面板，适合“找回刚才复制过的内容”场景。
- 剪贴板历史顶部说明更明确：双击是复制并关闭，Enter/粘贴按钮是粘贴并关闭，复制按钮只复制不关闭。
- 复制成功提示改为“文字/图片已复制到系统剪贴板”，小白更容易确认结果。
- 删除剪贴板历史记录前新增确认弹窗，说明只会删除本机历史，不影响当前系统剪贴板内容。
- 修复手势执行“粘贴 / 粘贴并回车”时只按下 Ctrl+V、但工位统计没有增加粘贴次数的问题。
- 剪贴板历史列表选中态也改为浅蓝高亮，整体视觉更一致。

### 验证

- dotnet test 通过，472 passed。

## GestureClip v0.6.0 Beta

### 重点更新

- 工位小熊升级为办公状态 Hub：顶部实时状态卡、今日收益与效率、摸鱼与休息、下班冲刺、今日报告。
- 新增手动摸鱼模式，记录本次/今日摸鱼时长与娱乐估算价值。
- 新增下班后自动报告，每天最多弹出一次，支持今天不再显示。
- 新增 PNG 分享卡，默认保存桌面，不上传、不自动分享、不包含剪贴板内容。
- HUD 联动工位状态，支持阶段变色和高频“复制 +5”合并提示。
- 托盘菜单新增工位小熊子菜单：Hub、摸鱼、报告、分享卡、休息提醒。
- 剪贴板历史新增“常显”按钮：默认失焦自动隐藏，选择常显后才置顶常驻。
- 设置页同步新增自动报告、摸鱼、下班冲刺、分享卡、HUD 状态文案、HUD 阶段变色等开关。
- 工位小熊 Hub 数据库迁移改为可重复检查，增强旧库恢复能力。

### 隐私与性能

- 报告和分享卡不包含剪贴板正文、图片内容、浏览器内容或敏感路径。
- 工位小熊统计全部本地保存，不上传。
- HUD 使用短缓存，托盘菜单避免同步阻塞，分享卡后台生成。

### 验证

- dotnet restore 通过。
- dotnet build 通过，0 警告 0 错误。
