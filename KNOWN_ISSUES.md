# KNOWN_ISSUES

GestureClip v0.6.29 Beta 当前已知问题。

## 已知问题

1. 当前 Beta 默认仍可能未签名（需配置 GESTURECLIP_SIGN_* 证书后才会签名）；Windows SmartScreen 可能提示未知发布者。
2. 管理员权限窗口中，普通权限运行的 GestureClip 可能无法稳定执行输入模拟（可尝试以管理员身份运行）。
3. 图片历史数量很大时，首次缩略图加载仍可能需要一点时间。
4. 覆盖更新前必须先退出程序，否则 `GestureClip.exe` 可能被占用。
5. 某些安全软件、密码管理器或浏览器隐私场景可能会限制剪贴板访问。
6. 诊断包会包含最近日志文件，但不包含剪贴板数据库和图片原始内容。上传前建议自行检查。
7. 批量复制：不能混选文字+图片；图片一次只能选一张（会提示友好文案）。
8. 「已复制」绿条仅本进程会话有效，重启后清空。

## 建议反馈时提供

- GestureClip 版本
- Windows 版本
- 复现步骤
- 截图或录屏
- 诊断信息或诊断包

不要上传敏感剪贴板内容、密码、token、cookie。
