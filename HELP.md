# HELP

## 打开剪贴板历史

默认快捷键：

```text
Ctrl + `
```

再次按下可以关闭剪贴板面板。

## 使用右键手势

按住鼠标右键，往一个方向画，松开右键执行动作。

常用例子：

- 上划：复制
- 下划：粘贴
- 下左：粘贴并回车
- 左划：后退
- 右划：前进

## R+L 组合手势

按住右键时，点一下左键，可以触发 R+L 组合手势。

注意：这不是把左键设为手势触发键。普通左键点击、拖拽、选中文本应该正常。

## 添加自定义手势

1. 打开设置。
2. 进入“手势绑定”。
3. 点击“开始设计”。
4. 在画板里画手势，或点击方向按钮。
5. 选择执行动作。
6. 点击“添加到手势列表”。

## 暂停剪贴板记录

可以在托盘菜单或设置页暂停。暂停后：

- 不再保存新的剪贴板内容
- 仍可打开和搜索旧历史
- 仍可粘贴旧历史

## 暂停鼠标手势

可以在托盘菜单或设置页暂停。暂停后：

- 不执行手势
- 不显示轨迹线
- 不显示 HUD
- 普通右键恢复正常

## 工位小熊

设置页进入“工位小熊”：

- 设置月薪
- 设置上下班时间
- 设置午休时间
- 设置工作日
- 查看今日工资、下班倒计时、发薪日倒计时

## 关闭过劳提醒

设置页进入“工位小熊”，关闭“开启过劳提醒”。

## 彻底退出程序

右键托盘图标，选择退出 GestureClip。

如果任务管理器还有 `GestureClip.exe`，说明退出异常，请反馈问题并附诊断信息。

## 导出诊断包

1. 打开设置。
2. 进入“诊断”。
3. 点击“导出诊断包”。
4. 上传前请确认包里没有你的敏感信息。

诊断包不包含剪贴板数据库和图片原始内容。
