# HELP

## 打开剪贴板历史

默认快捷键：

```text
Ctrl + `
```

再次按下可以关闭剪贴板面板。

### 剪贴板面板内快捷操作

| 操作 | 快捷键 / 入口 |
| --- | --- |
| 回到顶部 / 最新 | `Home` 或「↑ 顶部」 |
| 滚到底部 | `End` |
| 重置视图（清搜索 / 回全部） | 顶栏「重置」；`Esc` 可先清搜索 |
| 复制（不关面板） | `Ctrl + C` |
| 复制为纯文本 | `Ctrl + Shift + C` |
| 粘贴并关闭 | `Enter` |
| 粘贴不关闭 | `Ctrl + Enter` |
| 删除 | `Delete`（可立刻 `Ctrl + Z` 撤销） |
| 快捷键速查 | `?` 或 `F1` |
| 分类 | `Ctrl + 1`～`5` |

## 打开快捷动作（牛马助手 MVP）

默认快捷键：

```text
Ctrl + Shift + Q
```

也可从托盘菜单「打开快捷动作」进入。热键可在设置页「剪贴板 → 快捷动作热键」修改。

用途：对当前剪贴板文本做本地处理，例如去空格、大小写、JSON 美化/压缩、URL 编解码。默认把结果写回系统剪贴板；也可预览或处理并粘贴。

- 只做本地确定性动作，不登录、不上传。
- 日志只记录动作 id 和长度，不记录剪贴板正文。
- 手势绑定里可选「快捷动作 / 文本处理」分类，例如「文本：JSON 美化」「打开快捷动作面板」。

## 使用右键手势

按住鼠标右键，往一个方向画，松开右键执行动作。

常用例子：

- 上划：复制
- 下划：粘贴
- 下左：粘贴并回车
- 左划：后退
- 右划：前进

## R+L 组合手势

按住右键时，点一下左键，可以触发 R+L 组合手势。

注意：这不是把左键设为手势触发键。普通左键点击、拖拽、选中文本应该正常。

## 左键增强动作

和 R+L 不同：你是在**已经开始画方向手势**时再点左键，会执行“增强动作”。

设置入口：

```text
设置 → 动作绑定 → 左键增强动作（可设置）
```

默认：

- 下划 + 左键 → 智能粘贴（干净/纯文本倾向）
- 上划 + 左键 → 全选

可以自己增删改，也可“恢复默认”。

## 添加自定义手势

1. 打开设置。
2. 进入“手势绑定”。
3. 点击“开始设计”。
4. 在画板里画手势，或点击方向按钮。
5. 选择执行动作。
6. 点击“添加到手势列表”。

## 暂停剪贴板记录

可以在托盘菜单或设置页暂停。暂停后：

- 不再保存新的剪贴板内容
- 仍可打开和搜索旧历史
- 仍可粘贴旧历史

## 暂停鼠标手势

可以在托盘菜单或设置页暂停。暂停后：

- 不执行手势
- 不显示轨迹线
- 不显示 HUD
- 普通右键恢复正常

## 工位小熊

设置页进入“工位小熊”，或托盘菜单打开 **工位小熊 Hub**：

- 首次进入若未填月薪，会看到 **30 秒开工配置**（月薪 + 上下班时间）。
- Hub 分区可折叠：收益、摸鱼、休息提醒、下班冲刺、今日报告、周报/月报、工作规则。
- 休息提醒可设 **每日/本周上限**，以及 **连续工作满 N 分钟** 才弹常规定时提醒。
- 托盘可看 **今日已赚摘要**、开关 **下班冲刺**、生成本周总结。
- 分享卡片可选 经典蓝 / 简洁白 / 吐槽风 / 数据风，生成后可打开文件夹。
- 文案会随上午/午后/下班前自动换语气；HUD 彩蛋与快捷动作 XP 可在工作规则里开关。
- 设置月薪、上下班、午休、工作日；查看今日工资、下班倒计时、发薪日倒计时。

## 关闭过劳提醒

设置页进入“工位小熊”，关闭“开启过劳提醒”。

## 彻底退出程序

右键托盘图标，选择退出 GestureClip。

如果任务管理器还有 `GestureClip.exe`，说明退出异常，请反馈问题并附诊断信息。

## 导出诊断包

1. 打开设置。
2. 进入“诊断”。
3. 点击“导出诊断包”。
4. 上传前请确认包里没有你的敏感信息。

诊断包不包含剪贴板数据库和图片原始内容。

## 工位小熊 Hub

从托盘菜单 `工位小熊` 打开 Hub，可查看实时状态、今日收益与效率、摸鱼模式、休息提醒、下班冲刺、今日牛马生存报告，并生成 PNG 分享卡片。

- HUD：动作后的轻量提示，不抢焦点，不阻挡点击。
- Hub：主动打开的状态中心，可交互、可生成报告和分享卡片。
- 摸鱼模式：只记录用户手动开始/结束的时长，不自动识别网页或应用内容。
- 休息提醒：轻量、可稍后、可今天不再提醒、可关闭，不做医学诊断，不强制锁屏或关闭程序。
- 今日报告：下班后每天最多自动弹出一次；可在 Hub 生成、关闭自动弹出或点“今天不再显示”。
- HUD 联动：可显示“复制成功 · 少点 +1”“复制 +5”等短文案，颜色可跟随工作阶段，也可在设置中关闭。
- 隐私：所有数据本地保存；报告和分享卡片不包含剪贴板正文、图片内容、浏览器内容或敏感路径。
