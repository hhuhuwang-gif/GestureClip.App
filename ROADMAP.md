# ROADMAP

GestureClip 的方向是：本地优先、轻量、顺手、有一点情绪价值的 Windows 办公效率助手。

## v0.6.x Beta：稳定和信任

- 修复 Beta 用户反馈的崩溃、卡顿和手势误触。
- 持续优化图片剪贴板兼容性和缩略图加载。
- 完善隐私说明和诊断包说明。
- 默认加入推荐隐私黑名单。
- 整理 README、CHANGELOG、HELP、UPDATE 等文档。
- 建立 GitHub Actions 自动 restore / build / test / package。

- 更清晰的下载页和 30 秒演示 GIF / 视频。——图文导览 docs/DEMO_30S.md；OCR/便携说明已补。
- 手势动作模板优化：办公 / 浏览 / 剪贴板一键模板已落地；常用话术与日期时间输入已落地。
- 手势配置导入 / 导出。——已落地（JSON）。
- 更细的应用级规则。——智能粘贴按进程覆盖 + 屏蔽名单增强已落地。

## 暂不做
